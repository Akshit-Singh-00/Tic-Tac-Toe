# Tic Tac Toe Game

A simple and interactive Tic Tac Toe game built using HTML, CSS, and JavaScript.

## Features
- Two player gameplay
- Winner detection
- Draw detection
- Restart button
- Responsive UI

## Tech Stack
- HTML
- CSS
- JavaScript
